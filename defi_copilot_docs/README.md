# DeFi Copilot

> The AI-powered trading assistant that lets anyone trade DeFi in plain English.

---

## What is DeFi Copilot?

DeFi Copilot is a non-custodial, AI-powered trading platform that removes the technical complexity of decentralized finance. Users connect their own wallet and interact entirely through natural language — the AI parses intent, finds the best route, and prepares the transaction. The user signs and approves. Funds never leave their control.

**Core promise:** "Swap $500 ETH at the best price" → done. No forms, no slippage math, no DEX expertise required.

## The Three Principles

1. **Non-custodial always** — the AI prepares, the user signs. Private keys never leave the user's device.
2. **AI is the interface** — no forms, no dropdowns. Everything is a conversation.
3. **Ship fast, validate first** — PWA web app before native mobile. One chain before many.
